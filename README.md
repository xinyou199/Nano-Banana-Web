# LhzyAI 绘图平台（基于 Nano-Banana）

本仓库实现了一个基于 Nano-Banana 的绘图平台网站（项目名：LhzyAI），支持微信小程序扫码登录、微信支付、后台任务处理和图片处理队列。该项目以 C# (.NET) 实现，包含前后端控制器、业务服务、任务队列与 R2 对象存储支持。
